# [PRD] 무형문화재 큐레이션 및 글로벌 체험 플랫폼: 킷다 (K-ItDa)

## 1. 프로젝트 개요
- **프로젝트명**: 킷다 (K-ItDa) - Korea Heritage Connection
- **비전**: "박물관 유리창 너머의 전통을 일상의 경험으로"
- **핵심 목표**: 
  - 국가유산청의 파편화된 데이터를 사용자 중심의 경험 데이터로 재구축.
  - 무형문화재 전수자들에게 디지털 판로와 브랜딩 기회 제공.
  - 글로벌 관광객 및 국내 MZ세대를 위한 '힙 트래디션(Hip-Tradition)' 경험 제공.

## 2. 기술 스택
- **Frontend**: Next.js (React), Tailwind CSS, Zustand/TanStack Query
- **Backend**: Node.js (NestJS), PostgreSQL, Redis
- **Infra**: AWS S3, GitHub Actions, Vercel
- **Third-party**: Portone (Payment), Google Maps, NextAuth.js
